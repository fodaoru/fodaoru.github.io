佛、道、儒

童蒙止观核心义理摘要

《修习止观坐禅法要》，隋·智顗述，为初心学人开止观之门。全文十意，次第井然，依具缘至证果，脉络清晰。引用原文以 粗体 标识。

总纲 · 止观双运

诸恶莫作，众善奉行，自净其意，是诸佛教。

止乃伏结之初门，观是断惑之正要。

法功用对应
止（奢摩他）伏结——爱养心识之善资禅定胜因
观（毗婆舍那）断惑——策发神解之妙术智慧由藉

当知此之二法，如车之双轮，鸟之两翼，若偏修习，即堕邪倒。

  • 偏修禅定不学智慧 → 名之曰 愚（如声闻定多，不见佛性）
  • 偏学智慧不修禅定 → 名之曰 狂（如十住菩萨慧多，见佛性而不明了）
  • 定慧力等 → 诸佛如来，了了见于佛性

若如是知者，止观法门实非浅。 —— 十意是初心登正道之阶梯。

第一 · 具缘（五缘）

修止观之前，先外具五种条件：

1. 持戒清净

根器状态穿喻
上品未造五逆，受戒以来清净护持，无所毁犯净衣易受染色
中品虽犯轻戒，如法忏悔，亦名清净衣曾垢腻，浣净可染
下品轻重多犯，依大乘教犹可灭除需如法忏

忏悔十法：明信因果 → 生重怖畏 → 深起惭愧 → 求灭罪方法 → 发露先罪 → 断相续心 → 起护法心 → 发大誓愿 → 常念十方佛 → 观罪性无生。

罪灭相：身心轻利、得好瑞梦、灵瑞异相、善心开发、禅境界现前、豁然解悟。

若犯重罪，摄心常坐，及诵大乘经，一切重罪悉皆消灭，诸禅三昧自然现前。

2. 衣食具足

项目三类标准
衣雪山大士（蔽形即足）→ 头陀粪扫三衣 → 寒国多许百一，须知量知足
食深山草果 → 常行乞食（破四邪命）→ 檀越送食 → 僧中洁净食

若过贪求积聚，则心乱妨道。

3. 得闲居静处

深山绝人处 → 头陀兰若（离聚落三四里）→ 清净伽蓝。

4. 息诸缘务

  • 息治生缘物
  • 息人间缘务（断绝人事往还）
  • 息工巧技术（医方、禁咒、卜相、书数算计）
  • 息学问缘务（读诵听学悉皆弃舍）

5. 近善知识

善知识作用
外护善知识经营供养，不相恼乱
同行善知识共修一道，互相劝发
教授善知识以禅定法门示教利喜

第二 · 诃欲（五欲）

五欲者，是世间色、声、香、味、触，常能诳惑一切凡夫，令生爱著。

欲典故作证过患
色欲频婆娑罗王身入敌国、优填王截五百仙人手足种种妙色，能令愚人见则生爱，作诸恶业
声欲五百仙人闻甄陀罗女歌声，失禅定，心醉狂乱闻即染著
香欲比丘在莲池闻花香，池神呵责偷香令诸结使皆起
味欲沙弥染著酪味，命终生酪中受虫身心生染著
触欲一角仙因触欲失神通，为淫女骑头障道业

此五欲者，得之转剧，如火益薪，其焰转炽。

五欲六喻：如狗啮枯骨、如鸟竞肉、如逆风执炬、如践毒蛇、如梦所得、如击石火。

一切众生，常为五欲所使，名欲奴仆。

无累无所欲，是名真涅槃。

第三 · 弃盖（五盖）

内意根中五种覆盖，障定慧不生：

1. 贪欲盖

欲为种种恼乱住处，若心著欲，无由近道。

诸欲求时苦，得时多怖畏，失时怀热恼，一切无乐处。

2. 嗔恚盖

嗔是失佛法之根本，堕恶道之因缘，法乐之冤家，善心之大贼。

九恼生嗔 → 嗔生怨 → 怨生恼 → 覆盖真心。

杀嗔则安乐，杀嗔则无忧，嗔为毒之根，嗔灭一切善。 当修慈忍以灭除之。

3. 睡眠盖

内心昏暗为睡，五情暗蔽为眠。睡眠如死，无所觉识，以不觉故，难可除灭。

如人被缚将去杀，灾害垂至安可眠！ 眠为大暗无所见，日日欺诳夺人明。

当警觉无常，减损睡眠。昏重者用禅镇杖却之。

4. 掉悔盖

掉（散乱）表现
身掉好游走戏谑，坐暂不安
口掉吟咏竞诤，无益戏论
心掉放逸攀缘，思维世间才技

掉散之人，如无钩醉象，穴鼻骆驼，不可禁制。

悔有二：掉后生悔；大作重罪常怀怖畏。悔箭入心，坚不可拔。

不应作而作，应作而不作，悔恼火所烧，后世堕恶道。

5. 疑盖

疑内容对治
疑自诸根暗钝，罪垢深重，非修道人勿自轻，宿世善根难测
疑师师仪貌不清净，岂能教我如臭皮囊中金，应生佛想
疑法犹豫不信，法不染心佛法之中，信为能入

如人在歧路，疑惑无所趣；诸法实相中，疑亦复如是。

五盖总摄

贪欲盖 = 贪毒，嗔恚盖 = 嗔毒，睡眠 + 疑 = 痴毒，掉悔 = 等分。除此五盖，即除一切不善之法。 如负债得脱、重病得差、恶贼中得免。

第四 · 调和（五事）

三界无别有，唯是一心作。若知心无性，则诸法不实。 如陶师调泥，如弹琴调弦，善调五事，三昧易生。

1. 调食

过过患
过饱气急心满，百脉不通，令心闭塞
过少身羸心悬，意虑不固
秽触之物心识昏迷
不宜之物动宿病，四大违反

身安则道隆，饮食知节量。

2. 调睡眠

眠是无明惑覆，不可纵之。 过眠废修丧失功夫，能令心暗昧，善根沉没。

初夜后夜，亦勿有废，无以睡眠因缘，令一生空过。 当念无常之火，早求自度。

3—5. 调身、调息、调心（三事合用）

分为 入、住、出 三相。

入定调三事

调身（七步）：安坐处 → 正脚（半跏/全跏）→ 解宽衣带 → 安手（左手置右手上）→ 正身（端直脊骨，勿曲勿耸）→ 正头颈（鼻与脐相对）→ 口吐浊气三口 → 闭口（唇齿相拄，舌向上腭）→ 闭眼（断外光）。举要言之，不宽不急，是身调相。

调息：息四相——风（有声）、喘（结滞不通）、气（出入不细）、息（出入绵绵，若存若亡）。

守风则散，守喘则结，守气则劳，守息则定。

调息三法：下著安心 → 宽放身体 → 想气遍毛孔出入通同无障。不涩不滑，是调息相。

调心：调伏乱想不令越逸；令沉浮宽急得所。

病相表现对治
沉相心中昏暗，头好低垂系念鼻端，令心住缘中
浮相心好飘动，身亦不安安心向下，系缘脐中
心急摄心过度，胸臆急痛宽放其心，想气下流
心宽心志散慢，口中涎流敛身急念，令心住缘中

不沉不浮，是心调相。

住中调三事

坐中随时觉察身、息、心调与不调，随不调者调适。令三事无相乖越，和融不二。

出定调三事

先放心异缘 → 开口放气 → 微微动身 → 动肩臂头颈 → 动二足 → 手摩毛孔 → 摩手令暖掩两眼 → 待身热歇方可出入。若不尔者，细法未散，令人头痛，百骨节僵。

进止有次第，粗细不相违，譬如善调马，欲往而欲去。

第五 · 方便行（五法）

法内涵
欲志愿好乐一切禅智法门。一切善法，欲为其本。
精进坚持禁戒，弃于五盖，初夜后夜专精不废，如钻火未热终不休息
念念世间欺诳可贱，念禅定尊重可贵，得定方能广度众生
巧慧筹量世间乐（少苦多，虚诳不实）vs 禅定乐（无漏无为，永离生死）
一心分明决定一心修行止观，心如金刚，天魔外道不能沮坏

非智不禅，非禅不智。

第六 · 正修行

分二：于坐中修 + 历缘对境修。

一、于坐中修（五意）

1. 对治初心粗乱修止观

三止：

止方法性质
系缘守境止系心鼻端、脐间等处，令心不散事相
制心止随心所起即便制之，不令驰散事相
体真止随心所念一切诸法，悉知皆从因缘生，无有自性，则心不取，妄念心息理体

一切诸法中，因缘空无主，息心达本源，故号为沙门。

三际求心：过去已灭，现在不住，未来未至，了不可得。

二观：

观方法
对治观不净观（治贪）、慈心观（治嗔）、界分别观（治著我）、数息观（治多寻思）
正观观诸法无相，因缘所生，因缘无性，即是实相。先了所观之境一切皆空，能观之心自然不起。

已解见空者，一切无想念。

2. 对治心沉浮病

心沉（暗塞多睡）→ 修观照了；心浮（轻躁不安）→ 修止。

3. 随便宜

止不效即试观，观不效即试止。如服药，随便宜调顺。

4. 对治定中细心

入定后若生贪著执为实有 → 知虚诳不实即修止；若犹著爱见 → 修观，观定中细心不可得。

5. 均齐定慧

定多慧少 → 修观破析；慧多定少（如风中灯）→ 修止得定心（如密室灯）。

二、历缘对境修

于一切时中常修定慧方便，当知是人必能通达一切佛法。

六缘（行住坐卧作语）+ 六尘境（色声香味触法），共十二事。

每一事皆同模式：

  • 修止：了知因缘所生一切法皆不可得，妄念心息
  • 修观：反观能行/住/坐/卧……之心，不见相貌，知毕竟空寂

六根对境核心要义：

眼见色时修止：若见顺情之色，不起贪爱。若见违情之色，不起嗔恼。若见非违非顺之色，不起无明及诸乱想，是名修止。 余五根类此。

若能于行住坐卧见闻觉知等一切处中修止观者，当知是人真修摩诃衍道。

第七 · 善根发

善根发相有二：

外善根发相

布施、持戒、孝顺、供养三宝、听学等，与魔境相滥，不细分别。

内善根发相（五种禅定善根）

善根发相
1. 息道身心泯然空寂，不见身心相貌；忽觉八触（痛痒冷暖轻重涩滑），身心安定，虚微悦豫
2. 不净观见他男女身死，膨胀烂坏，白骨狼藉；或内身不净，白骨节节相拄——厌患五欲，不著我人
3. 慈心忽然发心慈念众生，亲人、中人、怨人乃至十方五道——心悦乐清净，颜色常和
4. 因缘观推寻三世无明行等因缘，不见人我，离断常，破执见
5. 念佛忽忆念佛功德相好不可思议，爱敬心生，三昧开发，身心快乐清净

制心一处，无事不办。

辨别真伪

邪伪禅相（当却之）真正禅相
身搔动、身重如压、轻欲飞、如缚、寒热不定空明清净
见异境、心暗蔽、起恶觉、散乱、躁动、悲思内心喜悦，憺然快乐
恶触身毛惊竖、大乐昏醉善心开发，信敬增长
若爱著即与鬼神法相应，好失心癫狂智鉴分明，身心柔软
厌患世间，无为无欲，出入自在

若知虚诳，正心不受不著，即当谢灭。

第八 · 觉知魔事

魔（梵音魔罗，秦言杀者）：夺行人功德之财，杀行人智慧之命。

四种魔

魔对治
烦恼魔自心正除遣
阴入界魔自心正除遣
死魔自心正除遣
鬼神魔须善识，分三种

鬼神魔三种

魔类相状对治
精魅按十二时辰化作虎兔龙蛇等形说其名字呵责，即谢灭
堆剔鬼如虫蝎缘人头面，击枥腋下，抱持，言语喧闹闭目阴骂，诵戒本/三皈五戒
恼魔化作五尘三种境界：顺情（可爱）、违情（可畏）、平等（平常）修止观

却魔二法

修止却之：悉知虚诳，不忧不怖，不取不舍，息心寂然，彼当自灭。

修观却之：反观能见之心，不见处所，彼何所恼。

知魔界如即佛界如，若魔界如、佛界如，一如无二如，则魔界无所舍，佛界无所取，魔界自然消灭。

当其幻化，愚人不了，心生惊怖及起贪著，因是心乱，失定发狂，自致其患，皆是行人无智受患，非魔所为。

十种魔军

欲 → 忧愁 → 饥渴 → 渴爱 → 睡眠 → 怖畏 → 疑悔 → 嗔恚 → 利养虚称 → 自高慢人。

若分别忆想，即是魔罗网；不动不分别，是则为法印。

除诸法实相，其余一切皆是魔事。

第九 · 治病

若善用心者，四百四病自然除差。若用心失所，四百四病因之发生。

病发相

病因症状
四大增损地大增（肿结沉重）、水大增（痰饮不消）、火大增（寒热壮热）、风大增（虚悬战掉）
五脏生患心（寒热头痛口燥）、肺（胀满鼻塞）、肝（忧愁嗔恚眼暗）、脾（身面游风痒痛）、肾（咽喉噎寒腹胀耳聋）

治病方法——止观二门

修止治病：

方法要点
安心止在病处心是果报之主，如王至处群贼迸散
止心丹田（脐下一寸）久守即多所治
常止心足下安心在下，四大调适
知诸法空无所有何为病本？所谓攀缘。云何断攀缘？谓心无所得。

修观治病：

  • 六种气：吹、呼、嘻、呵、嘘、呬（心呵、肾吹、脾呼、肺呬、肝嘘、三焦嘻）
  • 十二种息：上息、下息、满息、焦息、增长息、灭坏息、暖息、冷息、冲息、持息、和息、补息
  • 假想观：患冷想火气起；患热想冷气起

当知止观二法，若人善得其意，则无病不治。

药石服饵、持咒治鬼病，修福忏悔治业报病。

治病十法

信 → 用 → 勤 → 常住缘中 → 别病因法 → 方便 → 久行 → 知取舍 → 持护 → 识遮障。

第十 · 证果——三止三观

三观次第

次第名称内涵对应智慧境界
一从假入空观了知一切诸法因缘虚假，体真止息一切智（慧眼）声闻辟支佛地，定多不见佛性
二从空入假观心性虽空，缘对亦能出生诸法，如幻化道种智（法眼）菩萨方便，慧多见佛性不明
三中道正观心性非空非假，不坏空假，息两边分别一切种智（佛眼）定慧力等，了了见佛性

因缘所生法，我说即是空，亦名为假名，亦名中道义。

证果之相

入中道正观 → 安住大乘 → 六根清净 → 入佛境界 → 成就念佛三昧 → 首楞严定 → 普现色身三昧 → 十方佛土教化众生 → 与普贤、文殊为等侣。

初发心时便成正觉，了达诸法真实之性，所有慧身不由他悟。

初发心菩萨即是佛，初心与后心无别。

结归止观

诸佛从心得解脱，心者清净名无垢。

初中后果皆不可思议，莫不皆约止观二门辨其究竟，并据定慧两法以明极果。
