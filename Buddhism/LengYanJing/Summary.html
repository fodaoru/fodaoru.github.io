佛、道、儒

楞严经核心义理摘要

依据十卷经文提炼，括号内注出处卷次，引用原文以 粗体 标识。

一、经题与缘起

经之五名（卷八）：大佛顶悉怛多般怛啰无上宝印十方如来清净海眼 / 救护亲因度脱阿难及此会中性比丘尼得菩提心入遍知海 / 如来密因修证了义 / 大方广妙莲华王十方佛母陀罗尼咒 / 灌顶章句诸菩萨万行首楞严。

缘起（卷一）：阿难被摩登伽女娑毗迦罗先梵天咒摄入淫席，将毁戒体。如来顶放光明，宣说神咒，敕文殊师利将咒往护，提奖阿难及摩登伽归。阿难悲泣悔恨多闻未全道力，殷勤启请 妙奢摩他、三摩、禅那最初方便，由此开演全经。

二、两种根本

一切众生从无始来，生死相续，皆由不知常住真心、性净明体。用诸妄想，此想不真，故有轮转。（卷一）

诸修行人不能得成无上菩提……皆由不知二种根本，错乱修习；犹如煮沙，欲成嘉馔，纵经尘劫，终不能得。（卷一）

名称内涵
一无始生死根本用攀缘心为自性者——妄识分别
二无始菩提涅槃元清净体识精元明，能生诸缘，缘所遗者——本觉真心

遗此本明，虽终日行，而不自觉，枉入诸趣。

三、七处征心

佛问阿难心目所在，阿难七次作答皆被破斥：

阿难计执佛破要点结论
心在身内若在身内，应先见脏腑，何不见？无有是处
心在身外身心相知不相离，一人食不饱众人无有是处
心潜伏根里（如琉璃合眼）若如琉璃合眼，何不见眼？无有是处
闭眼见暗名见内暗若在眼前，云何成内？无有是处
随所合处心则随有心若无体则无所合；若有体，出自内还是外？无有是处
心在中间根尘相对，中在何处？兼二不成无有是处
一切无著名为心有无俱不著方名无著，执无著即有着无有是处

核心开示（卷一）：如来常说诸法所生唯心所现，一切因果世界微尘因心成体。

若离前尘有分别性，即真汝心。若分别性离尘无体，斯则前尘分别影事。尘非常住，若变灭时，此心则同龟毛、兔角。

四、见性不动——客尘二义

佛屈指开合、放光摇首，令阿难辨动与不动（卷一）：

  • 佛手有开有合，见性无开无合
  • 阿难头自动摇，见性尚无有止，谁为摇动

以摇动者名之为尘，以不住者名之为客。 ——客尘即妄，见性即真。

㤭陈那因悟客尘二字成圣果：行客不住名客，主人常住名主；空中尘摇动，虚空寂然不动。结论（卷一）：

云何汝今以动为身，以动为境，从始洎终，念念生灭，遗失真性，颠倒行事；性心失真，认物为己，轮回是中，自取流转。

见性是心非眼（卷一）：无手人拳灭，无眼人见未全无（盲人见暗即见未亏）。灯能显色，如是见者是眼非灯；眼能显色，如是见性是心非眼。

五、见性无还——八还辨见（卷二）

佛以讲堂八种变化相示阿难：明还日轮、暗还黑月、通还户牖、壅还墙宇、缘还分别、顽虚还空、郁𡋯还尘、清明还霁。

诸可还者，自然非汝。不汝还者非汝而谁？

一切现象皆有所还之处，唯有 能见之性无可还，即是真心。

六、见性不灭——波斯匿王观河（卷二）

佛问波斯匿王：三岁见恒河，至年六十二，观河之见可曾有变？

汝面虽皱，而此见精，性未曾皱。皱者为变，不皱非变；变者受灭，彼不变者，元无生灭，云何于中受汝生死？

色身念念变灭，能见之性不生不灭。

七、颠倒与悟（卷二）

不知色身，外洎山河虚空大地，咸是妙明真心中物。譬如澄清百千大海，弃之，唯认一浮沤体。

晦昧为空，空晦暗中，结暗为色，色杂妄想，想相为身。聚缘内摇，趣外奔逸，昏扰扰相，以为心性。 ——此颠倒之由。

若能转物，则同如来，身心圆明，不动道场，于一毛端遍能含受十方国土。

八、超越因缘与自然（卷二）

当知如是精觉妙明，非因非缘，亦非自然，非不自然，无非不非，无是非是，离一切相，即一切法。

见明之时见非是明，见暗之时见非是暗；见见之时，见非是见，见犹离见，见不能及。

九、两种妄见（卷二）

类别比喻含义
别业妄见目有赤眚，夜见灯光圆影个人业感，唯眚者见
同分妄见一国人同见不祥境界共业所感，同分显现

见与见缘并所想相，如虚空华本无所有。此见及缘元是菩提妙净明体，云何于中有是、非是？

十、五阴本如来藏妙真如性（卷二—卷三）

阴譬喻结论
色阴瞪目发劳，见空中华色阴虚妄，本非因缘，非自然性
受阴二手相摩，妄生涩滑冷热受阴虚妄，本非因缘，非自然性
想阴谈梅口水出，思崖足酸涩想阴虚妄，本非因缘，非自然性
行阴暴流波浪相续行阴虚妄，本非因缘，非自然性
识阴频伽瓶塞孔擎空远行识阴虚妄，本非因缘，非自然性

因缘和合，虚妄有生；因缘别离，虚妄名灭。生灭去来，本如来藏常住妙明，不动周圆妙真如性。性真常中，求于去来、迷悟、死生，了无所得。

十一、六入、十二处、十八界皆如来藏（卷三）

同理破析眼耳鼻舌身意六入、六根六尘十二处、根尘识十八界，皆 虚妄无体，本非因缘非自然，俱为如来藏妙真如性。

十二、七大圆融——如来藏随缘（卷三）

破四大和合之说，揭露七大实相：

大核心句
地如来藏中，性色真空，性空真色，清净本然，周遍法界
火性火真空，性空真火
水性水真空，性空真水
风性风真空，性空真风
空性觉真空，性空真觉
见性见觉明，觉精明见
识性识明知，觉明真识

总公式：性 X 真空，性空真 X，清净本然，周遍法界；随众生心，应所知量，循业发现。世间无知，惑为因缘及自然性，皆是识心分别计度，但有言说，都无实义。

阿难悟后说偈（卷三）：

妙湛总持不动尊，首楞严王世希有。 销我亿劫颠倒想，不历僧祇获法身。 …… 如一众生未成佛，终不于此取泥洹。

十三、世界生成三续（卷四）

性觉必明，妄为明觉。觉非所明，因明立所；所既妄立，生汝妄能。无同异中，炽然成异；异彼所异，因异立同；同异发明，因此复立无同无异。如是扰乱，相待生劳，劳久发尘，自相浑浊，由是引起尘劳烦恼。起为世界，静成虚空。虚空为同，世界为异，彼无同异，真有为法。

三续成因
世界相续觉明空昧相待成摇 → 风轮→金轮→火→水轮 → 交妄发生递相为种
众生相续异见成憎，同想成爱 → 流爱为种，纳想为胎 → 胎卵湿化
业果相续欲贪→杀贪→盗贪；汝负我命，我还汝债；汝爱我心，我怜汝色；杀盗淫三为根本

十四、如来藏三义（卷四）

层面经文
空如来藏非心非空，非地非水，非风非火，非眼非耳鼻舌身意……
不空如来藏即心即空，即地即水，即风即火，即眼即耳鼻舌身意……
离即离非离即离非，是即非即

歇即菩提（卷四）：狂性自歇，歇即菩提。胜净明心，本周法界，不从人得。

菩提心生，生灭心灭，此但生灭。灭生俱尽，无功用道。（卷四）

十五、二决定义与五浊（卷四）

第一义：应以不生灭心为本修因。若以生灭心求佛乘不生不灭，无有是处。

第二义：审烦恼根本——眼、耳、鼻、舌及与身、心，六为贼媒，自劫家宝。

五浊：劫浊 → 见浊 → 烦恼浊 → 众生浊 → 命浊。如清水投土尘成浊，五浊亦复如是。

十六、六根功德与一门深入（卷四）

根功德原因
眼800后暗前明，三分阙一
耳1200十方无遗，圆满
鼻800有出有入阙中交

随汝详择其可入者……十方如来于十八界一一修行，皆得圆满……但汝下劣，未能于中圆自在慧，故我宣扬，令汝但于一门深入；入一无妄，彼六知根一时清净。

十七、六根生成与解结（卷四—卷五）

六根生成机制（卷四）： 由明暗等二种相形，于妙圆中，黏湛发见…… 由动静、通塞、恬变、离合、生灭等二种妄尘，黏湛发见/听/嗅/尝/觉/知，结色/声/香/味/触/法成根。

解脱原理（卷四）：汝但不循动静、合离、恬变、通塞、生灭、明暗，如是十二诸有为相；随拔一根，脱黏内伏，伏归元真，发本明耀。

击钟验常（卷四）：声尘有生灭，闻性不生灭。知有知无，自是声尘或无或有，岂彼闻性为汝有无？ 如重睡人梦中闻舂捣声，其形虽寐，闻性不昏。

若弃生灭，守于真常，常光现前，根尘识心应时销落。

十八、根尘同源 缚脱无二（卷五）

十方如来异口同音告阿难：

使汝轮转，生死结根，唯汝六根，更无他物。令汝速证安乐解脱、寂静、妙常，亦汝六根，更非他物。

根尘同源，缚脱无二，识性虚妄，犹如空华。知见立知，即无明本；知见无见，斯即涅槃。

佛说偈（卷五）：

真性有为空，缘生故如幻； 无为无起灭，不实如空华。 言妄显诸真，妄真同二妄。 …… 自心取自心，非幻成幻法； 不取无非幻，非幻尚不生，幻法云何立？ 是名妙莲华，金刚王宝觉； 如幻三摩提，弹指超无学。

解结喻（卷五）：巾本一条，绾成六结，解须从结心。六根同体而结有次第，解亦次第：先得人空 → 成法解脱 → 俱空不生 → 得无生忍。

二十、观世音菩萨耳根圆通（卷六）——重点

修行次第

初于闻中，入流亡所，所入既寂，动静二相了然不生。 如是渐增，闻所闻尽，尽闻不住；觉所觉空。 空觉极圆，空所空灭；生灭既灭，寂灭现前。

阶段关键解义
1入流亡所反闻照性，不逐声尘，能所双亡
2闻所闻尽能闻与所闻俱寂
3觉所觉空觉照与所觉皆空
4空所空灭空相亦空
5寂灭现前生灭灭已，寂灭为乐

二种殊胜

  • 上与十方诸佛同一慈力
  • 下与六道众生同一悲仰

三十二应身

随类化现佛、独觉、缘觉、声闻、梵王、帝释、自在天、大自在天、天大将军、四天王、太子、人王、长者、居士、宰官、婆罗门、比丘、比丘尼、优婆塞、优婆夷、女主、童男、童女、天龙药叉乾闼婆阿修罗紧那罗摩呼罗伽人身非人身等，说法度脱。

十四无畏

灾害不入、水火灾灭、刀兵段坏、恶鬼远离、枷锁解脱、贼不能劫、远离贪嗔痴、求男得男、求女得女、持名功德等同于持六十二恒河沙诸法王子名号。

四不思议

现众多妙容及神咒、现一一形诵一一咒施无畏、舍身珍宝供养、求妻得妻，求子得子……乃至求大涅槃得大涅槃。

文殊菩萨拣择圆通（卷六）

此方真教体，清净在音闻。欲取三摩提，实以闻中入。

大众及阿难，旋汝倒闻机，反闻闻自性，性成无上道，圆通实如是。 此是微尘佛，一路涅槃门。

二十一、三无漏学（卷六）

摄心为戒，因戒生定，因定发慧，是则名为三无漏学。

明诲核心教诫精要引文

  • 十二由旬成结界地，恶灾不入
  • 金刚藏王菩萨昼夜随侍

二十六、七趣轮回（卷八—卷九）

内分与外分

  • 内分（情）：爱染发起妄情，诸爱虽别，流结是同，润湿不升，自然从坠
  • 外分（想）：渴仰发明虚想，诸想虽别，轻举是同，飞动不沉，自然超越

临终去相

纯想即飞，必生天上。情少想多，轻举非远。情想均等，不飞不坠，生于人间。情多想少，流入横生。纯情即沉，入阿鼻狱。

地狱十因六报

十习因：淫（欲火）→贪（贪水）→慢（痴水）→嗔（刀剑）→诈（豺狼）→诳（蛇虺）→怨（鸩酒）→见（毒壑）→枉（霹雳）→讼（高山履海）。

六交报：见报（猛火满十方界）→闻报（波涛没溺）→嗅报（毒气充塞）→味报（铁网猛焰）→触报（大山四面来合）→思报（恶风吹坏国土）。皆为六根交作之恶果。

如是恶业，本自发明，非从天降，亦非地出，亦非人与，自妄所招，还自来受。菩提心中，皆为浮虚妄想凝结。

鬼道（十种）

怪鬼 → 魃鬼 → 魅鬼 → 蛊毒鬼 → 疠鬼 → 饿鬼 → 魇鬼 → 魍魉鬼 → 役使鬼 → 传送鬼。

畜生道（十种）

各由鬼业报尽转生，从枭类、咎征类、狐类……至循类。

人道（十种）

由畜生酬足复形为人，各以宿习参入顽类、异类、庸类……至达类。

十种仙

坚固服饵（地行）→草木（飞行）→金石（游行）→动止（空行）→津液（天行）→精色（通行）→咒禁（道行）→思念（照行）→交遘（精行）→变化（绝行）。斯亦轮回妄想流转，不修三昧，报尽还来散入诸趣。

诸天

  • 欲界六天：四天王天 → 忉利天 → 须焰摩天 → 兜率陀天 → 乐变化天 → 他化自在天
  • 色界：初禅三天（梵众/梵辅/大梵） → 二禅三天（少光/无量光/光音） → 三禅三天（少净/无量净/遍净） → 四禅四天（福生/福爱/广果/无想） → 五不还天（无烦/无热/善见/善现/色究竟）
  • 无色界：空处 → 识处 → 无所有处 → 非想非非想处
  • 修罗：卵生鬼摄 → 胎生人摄 → 化生天摄 → 湿生畜摄

精研七趣，皆是昏沉诸有为相，妄想受生，妄想随业；于妙圆明无作本心，皆如空华，元无所著。

二十七、五十阴魔（卷九—卷十）

行人修三摩提，自色阴尽至识阴尽，经历五十种境界。每一阴各有十种魔境，关键在于 不作圣心名善境界，若作圣解即受群邪。

阴根本妄想十境要点
色阴坚固妄想身能出碍→内彻拾虫→空中闻法→见佛国土→虚空成宝色→暗室见物→身同草木→见佛国地狱→遥见远方→妄见善知识形变
受阴虚明妄想悲魔→狂魔→忆魔→易知足魔→常忧愁魔→好喜乐魔→大我慢魔→好轻清魔→空魔→欲魔
想阴融通妄想贪求善巧（怪鬼）→贪求经历（魃鬼）→贪求契合（魅鬼）→贪求辨析（蛊毒鬼）→贪求冥感（疠鬼）→贪求静谧（大力鬼）→贪求宿命（山川鬼神）→贪求神通（山精海精）→贪求深空（日月薄蚀精气）→贪求长寿（自在天魔）
行阴幽隐妄想二无因论→四遍常论→四颠倒常无常论→四有边论→四矫乱论→死后有相→死后无相→死后俱非→七断灭→五现涅槃
识阴颠倒微细精想因所因执→能非能执→常非常执→知无知执→生无生执→归无归执→贪非贪执→真无真执→定性声闻→定性辟支

当处禅那，觉悟无惑，则彼魔事，无奈汝何。（卷九）

末法救度：不作圣心，名善境界；若作圣解，即受群邪。

二十八、五阴五妄想（卷十）

五阴本因，同是妄想。

阴妄想名简释
色阴坚固妄想汝体因父母想生，汝心非想则不能来
受阴虚明妄想顺益违损二现驱驰
想阴融通妄想寤即想心，寐为诸梦，摇动妄情
行阴幽隐妄想念念迁流，日夜相代，曾无觉悟
识阴颠倒微细精想湛非真湛，如急流水，流急不见，非是无流

二十九、总结要义（卷十）

五阴边际

惟色与空是色边际，惟触及离是受边际，惟记与忘是想边际，惟灭与生是行边际，湛入合湛归识边际。

修证总纲

生因识有，灭从色除。

理则顿悟，乘悟并消；事非顿除，因次第尽。

经之功德

若人一念将此法门于末劫中开示未学，四重十波罗夷罪障应念消灭，变地狱苦因成安乐国。

经之别名

大佛顶如来密因修证了义诸菩萨万行首楞严经。

佛说此经已，比丘、比丘尼、优婆塞、优婆夷，一切世间天、人、阿修罗，及诸他方菩萨、二乘、圣仙童子，并初发心大力鬼神，皆大欢喜，作礼而去。
