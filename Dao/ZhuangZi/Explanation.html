佛、道、儒

《庄子》解释

德充符

夫保始之征，不惧之实。勇士一人，雄入于九军。将求名而能自要者，而犹若是。而况官天地，府万物，直寓六骸，象耳目，一知之所知，而心未尝死者乎！

为名利所困的勇士，都能以无惧的心态入于九军。何况官天地、府万物，已经完全无我的得道之人。

    

天地

天地虽大，其化均也；万物虽多，其治一也；人卒虽众，其主君也。

天地万物随天道而化，民众随君道而行。 （仍然是强调无为，顺应天道。并且后文也强调上古之君也都是行无为之治。）

    

无为为之之谓天，无为言之之谓德，爱人利物之谓仁，不同同之之谓大，行不崖异之谓宽，有万不同之谓富。故执德之谓纪，德成之谓立，循于道之谓备，不以物挫志之谓完。

无为即无造作之心，顺应天道。其在天为道，在人为德。

    

黄帝遗玄珠，只有罔寻到了。罔即代表了离形去智的状态，玄珠代表了”天府“、”本性“。

    

在尧与许由的对话中，啮缺是一个世俗之中的聪明、机敏人，但其未达真知，未得真道。其乘人而无天、四顾而物应。因此，许由认为其可以治一方，不足以治天下。

    

在尧与华封人的对话中，尧说”多男子则多惧，富则多事，寿则多辱。“，看似有德之言，实则仍是造作之心而答的，并非真正天德。

    

禹与伯成子高的故事中，说明禹之有为之治，导致”德自此哀，刑自此立，后世之乱自此始矣。“

    

泰初有无，无有无名；一之所起，有一而未形。

道生一的过程。

同乃虚，虚乃大。合喙鸣；喙鸣合，与天地为合。

混同万物，则心境虚明，则容纳万物。此时言语如同鸟叫一般自然流淌。自然流淌即与天道相合。

    

孔子求道老子。名家这种”然不然，可不可“，善于辩论的人，并不能被称为圣人。其也只是有些聪明机智，并且这些人会被其聪明所困。就如猎狗因善长捕猎被困，所谓”聪明人“也会因聪明而”劳形怵心“。

    

将闾葂与季彻的对话中，说明了人道与天道相比，犹如螳蜋之于车辙。劝鲁君行有为之法，如果不顺天道，则其只是螳臂当车。（仍然是批判有为之治）

    

子贡与为圃者的故事，主要表达”有机事者必有机心“。机心即有为、造作之心。仍然是鼓励人反朴归真，存养心性。

    

谆芒与苑风对话中，”大壑“表示自性。（注焉而不满，酌焉而不竭，此在齐物论中也有出现，用来描述”葆光“，也即自性）

官施而不失其宜，拔举而不失其能，毕见情事而行其所为，行言自为而天下化，手挠顾指，四方之民莫不俱至。

圣治的描述，设置官职，但只按天道自然行事。言行亦符合自然之道，天下万物自然而化。举手示意，四方百姓没有不来投奔的。

神人：虚空粉碎（与形灭亡）、空明（照旷）

    

门无鬼与赤张满稽，圣人”治未病“，即防患于未然。即使被称为圣人的舜治理天下，也是等天下出了问题才治理。（即秃而施髢，病而求医）

要想治未病，则就要”不尚贤，使民不争；不贵难得之货，使民不为盗；不见可欲，使民心不乱“。

端正而不知以为义，相爱而不知以为仁。

端正而不知才能称为义。一旦知而为”端正“，则又会被异化，并不是发自本心而做了。

    

民众在有为之治下，民心乱，大惑且大愚。慢慢的，整个天下人都是大惑之人了。此时再宣扬高言、大声，则不能入于众人之心了，因为”惑者胜也“

世俗之所谓然而然之，所谓善而善之，则不谓之道谀之人也。

世俗人认可的，自己也认可；世俗人认为善的，自己也认为善的。这样也不并认为自己是阿谀之人。（其实是阿谀世俗之众人）

    

百年之木，破为牺尊，不管是牺尊还是剩余被丢弃的残枝，其于失性一也。（骈拇篇亦有”失性一也“的论述。）

    

天道

归根曰静。不欲以静，天下将自定。清静为天下正。

圣人之静也，非曰静也善，故静也，万物无足以铙(náo)心者，故静也。

圣人心静，并非因静所谓”好“才静，而是万物都不足扰乱圣人之心，自然而静。

水静犹明，而况精神！圣人之心静乎，天地之鉴也，万物之镜也。

圣人心静，以此映照天地万物。

    

本在于上，末在于下；要在于主，详在于臣。三军五兵之运，德之末也；赏罚利害，五刑之辟，教之末也；

三军五兵、赏罚利害，都只是细之未节，非本非要。（非一）

天道有四时之序，春夏先，秋冬后。圣人取象，亦设人道之序。君先而臣从，父先而子从。

古之谓大道者，五变而形名可举，九变而赏罚可言也。

五变、九变，是前文所说的几个次序：

先明天而道德次之，道德已明而仁义次之，仁义已明而分守次之。分守已明而形名次之，形名已明而因任次之，因任已明而原省次之，原省已明而是非次之，是非已明而赏罚次之。

即：天 -> 道德 -> 仁义 -> 分守 -> 形名 -> 因任 -> 原省 -> 是非 -> 赏罚。

    

尧曰：​“吾不敖无告，不废穷民，苦死者，嘉孺子而哀妇人。此吾所以用心已。​”

不怠慢孤苦无依之人，不抛弃穷苦之人，悲悯死者，善待孩子，哀怜妇人。

    

孔子求道于老子，宣扬自己的理念，仍然被老子否定了。因为仁义非根本。天道为宗，圣人从之。

    

鼠壤有余蔬而弃妹之者，不仁也，生熟不尽于前，而积敛无崖。

鼠洞边有剩余的蔬菜，却弃之不顾，这是不仁！生熟食物堆积于前，却还聚敛不止。（食物已经够了却还在积聚）

士成绮问道老子时，老子之所以”昔者子呼我牛也而谓之牛；呼我马也而谓之马“，是因为”名者，实之宾也“。

    

轮扁与齐桓公对话中，也是表达了贱至贵的阶次：书 -> 语 -> 意 -> 不可言。

道不可言，书中所述之道，被轮扁认为是”古人之糟魄已“。

    

天运

以敬孝易，以爱孝难；以爱孝易，以忘亲难；忘亲易，使亲忘我难；使亲忘我易，兼忘天下难；兼忘天下易，使天下兼忘我难。

形式上的尊敬来孝容易，但以发自内心的爱来孝就比较难了； (传习录中有类似表述)

以爱来驱动孝容易，以自然而孝难；（即天地篇中”孝子不谀其亲，亲之所言而然，所行而善“）

自己自然而孝容易，但使父母也忘记父子关系难；

天下亦然。（至仁无亲）

夫孝悌仁义，忠信贞廉，此皆自勉以役其德者也，不足多也。

孝悌仁义等，都是自我约束的，是用来奴役天德的，不应该推崇。

至贵，国爵并焉；至富，国财并焉；至愿，名誉并焉。是以道不渝。

与”至仁无亲“对应的，亦可表述为”至贵无爵，至富无财“。内在本性所发，无出其上者。

    

刍狗，随时而变。祭祀前，盛以箧衍，巾以文绣。祭祀后，行者践其首脊，苏者取而爨之。

礼法也是如此，不同时，不同势。不能僵硬的去照搬古之礼法。但孔子执著于周礼，犹如推舟于陆，因此才遭受陈蔡之困等。

    

中无主而不止，外无正而不行。由中出者，不受于外，圣人不出；由外入者，无主于中，圣人不隐。

内心没有对“道”的真正领悟，将”道“呈现给他，他也拿不住。外在环境不正，则”天道“不可行于世。

内心有道了，但外在环境无法接纳时，则圣人不会将其展示出来；由外部传达至心中的道，如果心中无法领悟，圣人也不会刻意将其隐藏起来。

    

孔子见老子，仁义乱人心。

使民心亲，民有为其亲杀其杀，而民不非也。

使百姓相亲，并且在亲属之间也有亲疏之别，人们同样也不会非议。 (礼记中有”明亲疏之杀“，此处杀是等级、差别的含义)

    

庚桑楚

简发而栉(zhì)，数米而炊，窃窃乎又何足以济世哉！

过于陷于鸡毛蒜皮的杂事，则就会忽略主线，就难以济世。（数着米粒下锅，一根一根的头发梳头。）

民之于利甚勤，子有杀父，臣有杀君，正昼为盗，日中穴阫(péi)。

民众为利所困，就会出现子杀父，臣杀君的行为。

南荣趎请入就舍，召其所好，去其所恶。十日自愁，复见老子。

召其所好，去其所恶。 应该理解为修善断恶。

十天下来，他自己感到非常愁苦。（因为恶念难以去除干净）

夫外韄(huò)者不可繁而捉，将内揵(jiàn)；内韄者不可缪而捉，将外揵；外内韄者，道德不能持，而况放道而行者乎！

被世俗之事所困的人，不可去尝试抓取、控制纷繁复杂的外物，应该关闭内六根，保养心神；

内心有所束缚的人，不可强制控制心神，应该关闭外六尘；

内外都被困的人，根本无法把持道德，又怎么能依道而行呢？

    

卫生之经，顺其自然，顺应万物(与物委蛇)，无拘无束(翛)，通达无碍(侗)，如婴儿一般。

更进一步：不以人物利害相撄，不相与为怪，不相与为谋，不相与为事。

再进一步：动不知所为，行不知所之，身若槁木之枝，而心若死灰。

    

徐无鬼

徐无鬼两次见魏武侯，分别说明了要忘我、去欲。当政者不要有为，不要为了自己私欲而劳民。

    

黄帝黄帝将见大隗乎具茨之山之中，小童所说“奚以异乎牧马者哉？亦去其害马者而已矣！”，说明治天下与牧马没什么区别，自然无为就好。（治世）

小童游于襄城之野、六合之外，养其性情（病少痊）。六合之外即精神境界。（养生）

    

知士无思虑之变则不乐，辩士无谈说之序则不乐，察士无凌谇(suì)之事则不乐，皆囿于物者也。

人会不自觉的困于外物。意识不断攀缘外物，导致“农夫无草莱之事则不比”。人闲不下来，不断追逐外物，而导致“驰其形性，潜之万物，终身不反，悲夫！”。这也就是老子说的“其行弥远，其知弥少”。

    

射者非前期而中，谓之善射，天下皆羿也，可乎？​

先画靶子后射箭，天下皆羿。人也都是先有立场再有是非、言语。与齐物论的“未成乎心而有是非，是今日适越而昔至也。” 表达同样的观点。

    

桓公与管仲对话中，鲍叔牙过于“正”，无法容忍不善，不可为相。（对“有为”的规则过于执著，就会行“有为”之事，会损害人民的本性）

而隰(xí)朋“上忘而下不畔，愧不若黄帝而哀不己若者”，即不媚上欺下，严以律己，宽以待人。惭愧自己的德行不如黄帝，对不如自己的人又充满哀悯。

“其于国有不闻也，其于家有不见也”，则说明隰朋并不“简发而栉(zhì)，数米而炊”，能容人容事，不会出现“人至察则无徒”的情况。

以贤临人，未有得人者也；以贤下人，未有不得人者也。

《四十二章经》中也言“有势不临难”。能使自己“下”于他人，则人都会来归顺。

    

狙“伐其巧，恃其便”以敖吴王，最终导致被杀死。因此“无以汝色骄人哉”，即要谦下，与前面“以贤下人”等有相关。

    

南伯子綦是对“我执”的进一步探察。一些微弱的我执，可能会让自己“先之”，彼故知之。在无意识中，自己跟前文的“狙”一样，伐其巧，恃其便。

    

九方歅看相，言子綦之子与国君同食以终其身，被子綦视为不祥。因为自己不曾从事仕途，并且不与之为食，不与之为谋，不与之为道。邀乐于天，邀食于地。怎么会与国君扯上关系？这种怪事，必定有怪行，不是祥事。（后面果然是被人砍去双脚卖给渠公看门）

(世人一味追求名利，而意识不到其背后的危险)

吾未尝为牧，而牂(zānɡ)生于奥；未尝好田，而鹑生于宎(yǎo)，若勿怪，何邪？

没有养羊，但母羊生于家里的西南角；不曾打猎，但鹌鹑生于东南角。如果不是怪事，怎么理解？

    

爱之则亲，利之则至，誉之则劝，致其所恶则散。

民众很容易聚，爱他们则他们也会亲近你，给他们利益则他们就会聚过来，夸他们就是开心勤勉，给他们所厌恶的东西就会散开。

爱利出乎仁义，捐仁义者寡，利仁义者众。夫仁义之行，唯且无诚，且假乎禽贪者器。

爱与利都是出于仁义，也就是控制民众的工具。愿意丢弃仁义的人少，而利用仁义的人多。 仁义之行不仅无诚，而且被像禽兽一样贪婪的人当作工具。

（仁义也被异化为了贪名夺利的工具）

    

暖姝者、濡需者、卷娄者，分别指自满的人、安于一时、一处的人、为物所困的人。

未得大道而自满的人，谓之暖姝者。

安于一处，意识不到自身所处环境的，谓之濡需者。网上流传的“不在乎大国崛起，只在乎小民尊严”就是濡需者。

一生都被物为困，劳苦终身的，谓之卷娄者。（卷娄即衰老背驼，劳苦终生而致）

神人恶众至，众至则不比，不比则不利也。

神人不喜欢众人跟随自己，跟随自己则不舒坦，则不利。

于蚁弃知，于鱼得计，于羊弃意。

于民众弃智，于圣人弃有为之心。然后如鲦鱼从容而游，自由自在。

古之真人，以天待人，不以人入天。古之真人，得之也生，失之也死；得之也死，失之也生。

以自然之道对待人事，不以妄为之心扰乱天性、自然。（人道顺应天道）

生即生，死即死。以目视目，以耳听耳，以心复心。
